Les API

sont des blocs de code déjà prêts qui permettent à un développeur d'implémenter des programmes qui seraient difficiles voire impossibles à implémenter sans elles.

Les API peuvent accéder à des données de l'environnement informatique, ou faire d'autres choses complexes.

Par exemple :
l'API de géolocalisation récupère des informations géographiques. C'est ainsi que Google Maps peut trouver votre position et la situer sur une carte.
